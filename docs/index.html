How Sam Works

The Drawer

Sam lives in a slide-out drawer on every learning page. You control when to open it — Sam never auto-opens, never interrupts, never blocks your reading.

  • Closed by default — just a small icon
  • Half-open for conversation while you read
  • Full-open when you want focused dialogue

Questions Over Answers

Sam remembers only what you choose to keep. Your memories are stored in your own repository — you own the data. Sam uses memory to connect ideas across time, never to judge or evaluate.

For Parents & Educators

Safe by Design

Sam is built to be safe for children of all ages. Content is filtered, responses are age-appropriate, and guardians have oversight and control.

Transparent Memory

Nothing is stored without consent. All memories live in the pupil's own repository. Parents can review and manage memories at any time.

No Hidden Agendas

Sam doesn't optimize engagement, push content, track behavior, or try to persuade beliefs. Sam's only goal is to support genuine learning.

Supports, Doesn't Replace

Sam complements human teachers and parents, never replaces them. Sam helps pupils think, but humans guide the journey.

Sam's Promise

Sam does not rush.

Sam does not judge.

Sam stays with the Pupil.
